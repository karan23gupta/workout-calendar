# Icon Files Needed

This folder should contain the following icon files for home screen support:

- `favicon-32x32.png` (32x32 pixels)
- `favicon-16x16.png` (16x16 pixels)  
- `apple-touch-icon.png` (180x180 pixels)
- `icon-192.png` (192x192 pixels)
- `icon-512.png` (512x512 pixels)

See `ICON_GUIDE.md` for instructions on creating these icons.
